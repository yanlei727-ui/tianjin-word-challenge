# 天津中考英语学习系统 - 路由说明

## 路由总览

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 系统主页，展示七大核心模块入口 |
| `/vocabulary` | 词汇中心 | 单词本、训练、复习入口汇总 |
| `/grammar` | 语法专项 | 语法知识点学习与练习 |
| `/reading` | 阅读理解 | 阅读文章与题目练习 |
| `/cloze` | 完形填空 | 完形填空练习 |
| `/writing` | 作文训练 | 写作题目与范文参考 |
| `/mistakes` | 错题本 | 跨模块错题统一管理 |
| `/plan` | 学习计划 | 学习任务管理与进度概览 |

## 词汇子路由（保留）

| 路由 | 页面 | 说明 |
|------|------|------|
| `/learn` | 单词学习 | 逐词学习模式 |
| `/wordlist` | 单词本 | 按模块浏览单词列表 |
| `/challenge` | 闯关模式 | 分关卡答题 |
| `/wrongbook` | 错词本 | 查看词汇错题 |
| `/chinese-challenge` | 释义训练 | 看中文写英文 |
| `/quick-review` | 快速识词 | 快速翻卡标记 |
| `/choice-quiz` | 选择练习 | 四选一练习 |
| `/favorites` | 重点词 | 收藏的重点词汇 |
| `/favorite-quiz` | 重点词训练 | 重点词专项练习 |

## 查询参数

| 参数 | 用于 | 说明 |
|------|------|------|
| `?module=noun` | 词汇相关页面 | 选择词库模块（noun/adj_adv/verb） |
| `?mode=wrong` | 练习页面 | 错词模式 |
| `?mode=all` | 练习页面 | 全部词汇模式 |
| `?start=10` | 学习页面 | 从指定位置开始 |

## SPA 路由注意

由于使用 BrowserRouter，直接访问子路由（如 `/vocabulary`）需要服务器支持 SPA fallback。Cloudflare Pages 已配置 _redirects 规则。
